The Acacia Strain:

A heavy metal band from Chicopee, Massachusetts established in 2001.

Slogan:

"Tune low; die slow."

This website:

  • is an excercise in web development, which serves as part of the webmaster's application for admission to The Holberton School.
  • was created because The Acacia Strain currently lacks an official website, relying exclusively on social media accounts for its web presence.
  • is designed to appear to be an official website with the exception of the "log" webpage, where the webmaster chronicles his experiences with the band.
